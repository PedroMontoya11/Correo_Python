Para poder ejecutar el script de Python necesitarás realizar los siguientes pasos:
1. Instalar Python desde la página oficial o escribir en la consola de comandos "python3" darle al Enter, se nos redirige a la tienda de Microsoft y le damos a Instalar.

2. Si deseamos realizar el envío del correo a una cierta hora necesitaremos contar con la biblioteca "schedule". Si no la tenemos instalada, lo hacemos ejecutando el comando pip install schedule en la terminal.

3. Insertamos los datos que queramos del correo en el código de Python (remitente, destinatario, mensaje...)

4. Creamos una contraseña de aplicación para utilizar el SMTP. Para ello vamos a Gestionar tu cuenta de Google y escribimos en el buscador "Contraseñas de aplicación". Una vez escribamos nuestra contraseña, se mostraran las contraseñas de aplicación que tenemos creadas. Escribimos el nombre de la aplicación y hacemos clic en "Crear".
Copiamos el código que nos aparezca en la ventana emergente y lo pegamos en la parte de Contraseña del script.

5. En el caso de utilizar el código para programar el envío del correo, especificamos la hora a la que queremos que se mande el correo.

6. Ejecutamos el código para verificar su función.
